# 墨迹投影 (InkLight) — 产品全景

---

## 第一部分：愿景与战略

### 1. 愿景声明 (Vision Statement)

让每个人都能用手写创造光 —— 打开手机浏览器写几个字，墙上的激光一笔一画还原出来，连书写顺序都一模一样。

### 2. 使命 (Mission)

InkLight 是一个开源 DIY 激光投影装置：用户在手机上用浏览器打开 Web App 手写输入，通过 Web Bluetooth API 将笔画数据实时发送到 ESP32 主控的激光振镜投影盒，用可更换的 AA 电池供电，在墙面上一笔一画还原书写过程的完整动画。

复现华为智慧投影大灯「随心画」的核心魔法感，以 **¥250-350 物料成本、可更换电池、口袋尺寸、无需安装任何 APP** 的极致体验实现。

### 3. 北极星指标 (North Star Metric)

- **指标名称**: 每周投影次数 (Weekly Projections Created)
- **当前基线**: N/A — 新产品
- **6 个月目标**: 50 活跃设备, 周均投影 200 次
- **12 个月目标**: 500 活跃设备, 周均投影 2000 次

### 4. 核心问题

#### 4.1 问题描述

华为问界 M9「随心画」功能让手写投影成为爆款内容，但这体验被锁在 50 万+ 的车里。现有 DIY 方案要么太贵（树莓派+DLP ¥1100+），要么输入体验差（在小屏幕上写字），要么依赖微信小程序审核流程。能不能做一个零门槛、零审核、零安装的方案？

Web Bluetooth API 正是答案 —— 用户打开浏览器就能配对设备发送数据，不需要下载 APP，不需要微信审核，一个 URL 就能用。

#### 4.2 现有解决方案及其不足

| 现有方案 | 不足 |
|----------|------|
| 华为智慧大灯「随心画」 | 需购车（¥539,800+） |
| 树莓派 DLP 触屏一体机 | ¥1100+，需海淘，内置锂电 |
| X-Laser 激光振镜 | 小屏书写，内置锂电 |
| 微信小程序方案 | 需审核，BLE 权限可能被拒 |
| Web Bluetooth API + PWA | ✅ 免审核免安装，但 iOS Safari 不支持 |

### 5. 价值主张

**一句话**: 打开浏览器写几个字，墙上的激光一笔一画还原出来。不装 APP，不审核，¥249。

**核心差异化**:
1. **免安装，一个 URL 就能用** — PWA 安装到桌面就是"伪原生 APP"
2. **笔画顺序回放 (魔法核心)** — 激光逐笔还原书写过程
3. **换电池，不充电** — 4 节 AA，没电秒换
4. **物料 ¥246** — 不到一顿火锅钱

### 6. 战略支柱

1. **零摩擦上手** — 网页打开即用，无需下载/注册/审核
2. **极简硬件** — ESP32 + 振镜 + 激光 + AA 电池，4 个螺丝
3. **开源生态** — MIT 协议，GitHub Pages 免费托管

### 7. 商业模式

- **收入模型**: 硬件套件 + 成品机
- **定价**: 套件 ¥99-149 / 成品 ¥249-349
- **目标客户**: B2C，科技爱好者、情侣礼物、亲子教育

### 8. 关键假设与验证

| 假设 | 验证方法 | 成功标准 | 如果错误 |
|------|---------|---------|---------|
| Web Bluetooth 体验可接受 | 5 台不同手机实测连接速度 | 连接成功率 ≥ 90%，≤ 3s | 降级为 React Native APP (仅此需开发) |
| 笔顺回放是魔法体验 | 拍短视频投小红书 | 点赞 > 1000 | 尝试其他差异化 |
| ¥350 内能做出来 | 采购核心物料最简验证 | 墙上清晰可见 | 纯软件 APP 方案 |
| iOS 用户不会因 Safari 不支持而流失 | 看用户平台分布 | iOS 占比 < 40% 或用户愿用第三方浏览器 | 补做独立 iOS APP |

> **Web Bluetooth 兼容性须知**: Chrome Android ✅ / Chrome Desktop ✅ / Edge ✅ / Opera ✅ / Samsung Internet ✅ / Safari iOS ❌ (需引导安装 Bluefy 等第三方浏览器)

### 31. 决策依据

- **Web Bluetooth API 而非微信小程序**: 免审核、免安装、即时更新。一个 URL 就是产品。唯一代价是 iOS Safari 不支持，但 iOS 用户可以用 Bluefy 浏览器或等 Apple 支持（已列入 WebKit 路线图）
- **双色激光而非 RGB**: V1.0 红色 (¥3) + 绿色 (¥15) 已经足够呈现手写质感，RGB 的蓝色激光贵且光路复杂
- **AA 电池而非锂电**: 使用场景低频，换电池体验远好于闲置后等充电
